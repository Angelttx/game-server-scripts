# Game Server Scripts

Automation scripts for game servers.

## Scripts Included

- 🎮 **Minecraft** - Server setup, backup, restart
- 🎯 **CS:GO** - Server management
- 🎰 **Rust** - Server automation
- 🌐 **Generic** - Works with most games

## Features

- Automatic backups
- Server restarts
- Player notifications
- Resource monitoring

## Usage

```bash
chmod +x minecraft-backup.sh
./minecraft-backup.sh
```
